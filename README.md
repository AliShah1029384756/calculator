# Calculator

A clean, iOS-style calculator built with vanilla HTML, CSS, and JavaScript.

## Features
- Basic arithmetic: addition, subtraction, multiplication, division
- Percentage and sign toggle (+/-)
- Keyboard support (numbers, operators, Enter, Backspace, Escape)
- Expression history shown above result
- Responsive design

## How to Run
Open `index.html` in any browser.

## Tech Stack
HTML · CSS · JavaScript
